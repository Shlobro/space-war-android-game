# Main Java Source Guide

## Purpose

`app/src/main/java` is the root of the Kotlin source tree for the Android app.

## Subfolders

- `com/example/cw/`: Application entry point plus the gameplay and theme packages. See `com/example/cw/cw_developer_guide.md`.

## Working In This Folder

- Add new gameplay packages below `com/example/cw/` and document the specific package where the behavior lives.
- Keep cross-package conventions here high level and leave feature details to child package guides.
